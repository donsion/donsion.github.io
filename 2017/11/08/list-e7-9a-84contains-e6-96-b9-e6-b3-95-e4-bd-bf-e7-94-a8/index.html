Java中List的contains()方法使用

应用场景举例：从数据库里面查出用户List列表，要判断一个List<User>里面是否包含User A；

描述：用默认的contains方法肯定是无法得到正确返回结果的，因为我们将用户id作为唯一标识。List的 contains(o)方法，系统会对list中的每个元素e调用o.equals(e)，方法，加入list中有n个元素，那么会调用n次o.equals(e)，只要有一次o.equals(e)返回了true，那么list.contains(o)返回true，否则返回false。所以我们要根据自己的业务需求来改写equals方法，如：

Class User{

    private Integer id;

    getXxx;
    setXxx;

    @Override
    public booelan equals(Object o){
         if (this == o) return true;
         if (o == null || getClass() != o.getClass()) return false;
         User other = (User) o;
       if(this.id.equals((User) o).id)) return true;
    }
}
